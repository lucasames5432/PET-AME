Roleta das Equações

Desafie seus conhecimentos em equações! Resolva equações do 1º e 2º grau e acumule prêmios. Quanto mais acertar, maior o prêmio! Mas cuidado: um erro e você perde o prêmio acumulado.

Dicas para Resolver Equações

1. Equações do 1º Grau (ax + b = 0):
Solução: x = -b/a, desde que a ≠ 0.

2. Equações do 2º Grau (ax² + bx + c = 0):
Utilize a fórmula de Bhaskara: Δ = b² - 4ac
x = (-b ± √Δ) / (2a)

3. Casos Especiais:

  • Δ > 0: Duas raízes reais e distintas
  • Δ = 0: Uma raiz real (raiz dupla)
  • Δ < 0: Não possui raízes reais

4. Formato de Resposta:

  • Para uma raiz: digite apenas o número (ex: 3, -2, 1/2)
  • Para duas raízes: separe por vírgula (ex: 2, 5 ou -3, 1)
  • Para nenhuma raiz real: digite "n"
  • Use frações: ex: 1/2, -3/4
Dica: Quanto maior a sequência de acertos, maior o prêmio! Comece com prêmio 1 e cada acerto multiplica o prêmio. Um erro reseta para 1.

Guia do Jogo - Roleta das Equações

Bem-vindo à Roleta das Equações! Este jogo testa seus conhecimentos em equações do 1º e 2º grau. Quanto mais você acerta, maior fica seu prêmio!

1. Como Jogar

Uma equação será exibida. Você deve determinar suas raízes reais e digitar no campo de resposta. Após digitar, clique em "Verificar Resposta". Se acertar, o prêmio aumenta! Se errar, o prêmio volta para 1 e uma nova equação é gerada.

2. Sistema de Pontuação

  • Prêmio Inicial: 1 ponto
  • Cada acerto: O prêmio dobra (1, 2, 4, 8, 16...)
  • Erro: Prêmio volta para 1
  • Objetivo: Conseguir a maior sequência de acertos possível!

3. Formatos de Resposta

  • Nenhuma raiz real: Digite n (minúsculo)
  • Uma raiz: Digite o número (ex: 3, -2.5, 1/2)
  • Duas raízes: Separe por vírgula (ex: 2, 5 ou -3, 1.5)

4. Tipos de Equações

O jogo gera aleatoriamente equações do 1º grau (quando a = 0) e do 2º grau (quando a ≠ 0). Todas as equações geradas garantem raízes reais (Δ ≥ 0 para quadráticas ou b ≠ 0 para lineares).

Dica Importante: Use a fórmula de Bhaskara para equações do 2º grau: x = [-b ± √(b² - 4ac)] / (2a). Para equações do 1º grau, isole o x: x = -c/b.

5. Exemplos

  • Equação: 2x² - 8x + 6 = 0 → Resposta: 1, 3
  • Equação: x² - 4x + 4 = 0 → Resposta: 2
  • Equação: 2x + 4 = 0 → Resposta: -2
  • Equação: x² + 1 = 0 → Resposta: n
